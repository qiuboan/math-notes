**说明**

1. 讲等比数列的求和问题，注意介绍符号。
2. 用 $S_n$ 代表前$n$ 项的和，即 $S_n = a_1 + a_2 + ... + a_n$


**证明思路**

1. 考虑 $S_n$ 和 $S_{n-1}$
2. 把 $S_n$ 和 $S_{n-1}$ 看成两个变量（类比 $x$ 和 $y$)
3. 想办法建立两个关于 $S_n$ 和 $S_{n-1}$ 的方程，然后解方程得到 $S_n$
4. 列一个方程是容易的。
5. 列第二个方程可以考虑提公因式，提取一个 $q$ 出来试试。
